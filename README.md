# Peliculasapp

This app is developed in **Angular** using **TMDB API**.

There are many components:

1. Home Component showing:
    * A SlideShow with latests movies. 
    * An infinite scroll to present lots of them.
    * Top Rated movies.

2. Search Component to look for your favorite movie.

3. Description Component with its crew.

To implement it you can type:

```
npm install
```
